
wiringPi README
===============

This is a modified WiringPi for Banana Pi. We call it WiringBPi. It is based on the original WiringPi for Raspberry Pi created by Drogon (http://wiringpi.com/). The modification is done by LeMaker (http://lemaker.org). You can donwload the WiringPi from:
https://github.com/LeMaker-Team/WiringBPi.git

Any examples and usages you can also refere to the materials on http://wiringpi.com/ .
 
Thanks!
